HTML:结构

  • HTML是一种超文本标记语言，不是一种编程语言

  • 主要用来实现静态页面，我们在页面看到的文本，图片，视频，声音，表格，链接等都是通过HTML语言描述的

  • HTML的是由标签组成，给不同的标签不同的属性，可以实现不同的网页效果。

  • 页面基本结构:

    <!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>
页面标题</title>
</head>
<body>
 
<h1>一个标题</h1>
 
<p>一个段落</p>
 
</body>
</html>

  • 参考手册: HTML 标签列表（功能排序） | 菜鸟教程

介绍

  • CSS 用于 HTML 文档中元素样式的定义
  • 使用 CSS 的目的就是让网页具有美观一致的页面

    border: 1px solid #bfbcbc;

CSS创建

  • 内部样式表: 在html文件头部(<head>)用<style>定义样式表, 例如:

    <head>
    <meta charset="UTF-8">
    <title>模拟登录</title>
    <style>
        #loginForm {
            width: 300px;
            margin: 0 auto;
            padding: 20px;
            border: 1px solid #6e6c6c;
            border-radius: 5px;
        }
        <!--省略其他选择器-->
    </style>
</head>

  • 外部样式表: 将样式表保存在外部文件中, 在html的中使用链接到样式表, 例如

    <head>
    <meta charset="UTF-8">
    <title>模拟登录</title>
    <link rel="stylesheet" href="style/test1.css">
</head>

    style/test1.css:

     #loginForm {
     width: 300px;
     margin: 0 auto;
     padding: 20px;
     border: 1px solid #6e6c6c;
     border-radius: 5px;
 }

input {
    width: 250px;
    padding: 10px;
    margin-bottom: 10px;
    border: 1px solid #bfbcbc;
    border-radius: 4px;
}

h1.textStyle {
    text-align: center;	
}

  • 内联样式: 在相关的标签内直接使用样式（style）属性, 例如

    <p style="color:sienna;margin-left:20px">这是一个段落。</p>

    慎用, 建议只在一个元素上使用一次时使用

  • 多重样式优先级: 内联 > 内部 > 外部 > 浏览器默认

JavaScript:行动

介绍

  • JavaScript是一门轻量级的脚本语言，是一种运行在客户端的脚本语言

    Script 是脚本的意思

    脚本语言：不需要编译，运行过程中由 js 解释器(js 引擎）逐行来进行解释并执行

  • 不同于Java程序运行在JVM中，JavaScript运行是直接运行在浏览器中

  • 主要使用来实现页面功能和业务逻辑

参考手册

  • JavaScript 和 HTML DOM 参考手册 | 菜鸟教程 (runoob.com)

基础作用

  • 直接写入 HTML 输出流

  • 对事件的响应

  • 改变 HTML 内容, 图像, 样式

  • 验证输入

JavaScript使用

  • HTML中的 JS 代码必须在<script>标签中, 可以在<head>或<body>部分中, 例如:

    <!DOCTYPE html>
<html lang="zh-CN">
    <head>
        <meta charset="utf-8">
        <title>learn01</title>
    </head>
    <body>
        <script>
            document.write("hello world!")
        </script>
    </body>
</html>

    效果:

  • 函数: 在触发某种事件的时候可以调用函数, 以实现某些交互效果, 例如:

    <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <script>
        function test() {
            document.getElementById("demo").innerHTML = "Hello world!";
        }
    </script>
</head>

<body>
    <p id="demo">
        一个段落
    </p>
    <button type="button" onclick="test()">
        点我
    </button>
</body>

</html>

    点击后, 页面会出现一些变化

  • 外部的JavaScript:在<script>标签的”src”属性中设置 JS 文件的路径, 例如:

    <script src="script/script01.js"></script>

补充: 正则表达式

  • 一种文本匹配工具, 用于匹配字符串

  • 在线测试工具: regex101: build, test, and debug regex

  • 参考手册:

    • 正则表达式
    • learn-regex

  • 语法:

    • 元字符:

      • ? : 前一个字符可有可无(出现1次或0次), 如ab?c可以匹配ac和abc

      • * : 前一个字符可以出现0次或任意次, 如ab*c匹配ac, abc, abbbbc

      • + : 前一个字符至少出现一次

      • {数字n} : 更加精确的次数匹配, 前一个字符出现n次

      • {n1, n2} : 出现n1~n2次

      • {n1,} : 出现n1次或n1次以上

        tip: *相当于{0,} , +相当于{1,} , ？相当于{0,1}

      • (str) : 群, 例如(ab){3}匹配ababab

      • [] : 字符种类, 匹配方括号内的任意字符

      • [^ ] : 否定的字符种类, 匹配除了方括号里的任意字符

      • ^ : 表示开头

      • $ : 表示末尾

    • 或运算: (str1|str2), 例如a (ab|cd)匹配a ab和a cd

    • 简写字符集:

      简写 描述
      . 除换行符外的所有字符
      \w 匹配所有字母数字，等同于 [a-zA-Z0-9_]
      \W 匹配所有非字母数字，即符号，等同于： [^\w]
      \d 匹配数字： [0-9]
      \D 匹配非数字： [^\d]
      \s 匹配所有空格字符，等同于： [\t\n\f\r\p{Z}]
      \S 匹配所有非空格字符： [^\s]
      \f 匹配一个换页符
      \n 匹配一个换行符
      \r 匹配一个回车符
      \t 匹配一个制表符
      \v 匹配一个垂直制表符
      \p 匹配 CR/LF（等同于 \r\n），用来匹配 DOS 行终止符

  • JS 中的正则表达式:

    • 语法: /正则表达式主体/修饰符(可选)
      例如: 

      var reg = /test/i

      test 是一个正则表达式主体 (用于检索)。

      i 是一个修饰符 (搜索不区分大小写)。

      修饰符:

      修饰符 描述
      i 执行对大小写不敏感的匹配。
      g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。
      m 执行多行匹配。

    • 使用:

      • search(): 返回第一个匹配项的索引(下标), 没有则返回-1, 例如:

        const str = 'Hello, World!'; 
const index1 = str.search('World'); // 返回 7

      • replace(): 返回被替换后的新字符串, 例如:

        const originalString = 'Hello, World! Hello, Universe!';
const regex = /Hello/g;
const newString = originalString.replace(regex, "Hi");// 返回: 'Hi, World! Hi, Universe!'

      • test(): 检测是否可以匹配, 返回true或false, 例如:

        var regex = /^1[3456789]\d{9}$/
result = regex.test('13873489827') //返回true

      • exec(): 执行搜索, 返回包含匹配的子字符串或null, 例如:

        const regex = /(\d+)-(\w+)/;
const inputString = '2024-January';

// 使用 exec() 方法执行正则表达式匹配
const result = regex.exec(inputString);

if (result) {
  console.log('匹配成功:', result[0]);       // 输出: '2024-January'
  console.log('第一个捕获组:', result[1]);  // 输出: '2024'
  console.log('第二个捕获组:', result[2]);  // 输出: 'January'
} else {
  console.log('未找到匹配项');
}
