前端与HTML

基本概念

请求方法

  • GET: 获取资源

  • POST: 提交资源

响应状态码
  • 1XX: 服务器收到请求, 需要继续操作
  • 2XX: 成功
  • 3XX: 重定向, (跳转)
  • 4XX: 用户的问题(客户端错误 语法错误), 常见:
    • 403: 访问被拒绝
    • 404: 资源不存在
  • 5XX: 服务器的问题

详细:HTTP 响应状态码 - HTTP | MDN (mozilla.org)

HTTP协议组成

  • 请求头(标头header): HTTP 标头（header） - HTTP | MDN

    HTTP 请求头各参数具体含义_http请求头包含哪些参数-CSDN博客

    查看请求头: F12进入开发者工具->网络->点击名称列查看更加详细信息

  • 请求行, 请求体, http响应

作业:

  • 内容1：在菜鸟教程或W3Sschool进行自主学习前端内容，HTML、JavaScript以及CSS，并记录在笔记中，笔记内容包括概念、代码以及效果截图，笔记情况需要检查。

  • 内容2：学习HTTP协议请求结构，把相关概念与学习情况记录在笔记中，需要检查。

  • 内容3：写一个提交表单进行模拟登录，用户账号是手机号，不需要后端接收，但登录需要配置JS去验证手机号格式输入。

  • 笔记导出成pdf或word或md格式，和模拟登录代码一起打包成压缩包进行提交。

  • 学习: python基础 -> 爬虫
