Python爬虫基础

爬虫概念

定义

  • 网络爬虫是伪装成客户端与服务端进行数据交互的程序。

应用

  • 数据采集, 搜索引擎, 模拟操作, 软件测试, 网络安全(信息轰炸, web漏洞扫描)

一般流程

会话技术:

Cookie(s)

  • 介绍: 指某些网站为了辨别用户身份、进行 session 跟踪而储存在用户本地终端上的数据, 实际由服务器发送给客户端, 可以理解为一个凭证, 服务器可以以此辨别用户身份

  • 爬虫中使用cookie

    • 好处: 能够访问登录界面
    • 坏处: 容易被反爬

Session

  • 含义: 在爬虫中, 指一个浏览器窗口从打开到关闭的这个期间
  • 目的: 在一个客户打开浏览器到关闭的这个期间, 发起的所有请求都可以被识别为同一个用户
  • 实现: 客户访问网站时, 生成一个包含SessionID的cookie, 用于服务器识别
  • Session是储存在服务器上的数据

爬虫基础

urllib库request模块

简单示例

  • 导入urllib请求模块:

    import urllib.request

  • 发送GET请求:

    r = urllib.request.urlopen('http://www.baidu.com')

  • 查看返回信息:

    print(r.read().decode('utf-8'))

    返回了网页源码

urlopen方法

  • urlopen方法有三个参数

    urllib.request.urlopen(url, data=None, [timeout, ]*)

  • data: 给post请求携带参数, 比如在模拟登录时传入用户名和密码

  • timeout: 设置请求超时时间

  • urlopen 默认是 Get 请求, 当我们传入参数它就为 Post 请求了

Request方法

  • Request方法有四个参数:

    urllib.request.Request(url, data=None, headers={}, method=None)

  • headers: 请求头信息

  • method: 设置请求方式

requests库发送请求

  • GET请求示例:

    import requests


r = requests.get("https://www.bilibili.com/")
print(r)
print(r.ok)

    输出(失败):

    <Response [412]>
False

  • 修改请求头, 模拟浏览器访问:

    import requests


headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/121.0.0.0 Safari/537.36 Edg/121.0.0.0"
}
r = requests.get("https://www.bilibili.com/", headers=headers)
print(r)
print(r.ok)

    输出(成功):

    <Response [200]>
True

正则表达式匹配

导入模块-re
  • import re
捕获组
  • 要获取的字段要用括号( )括起来
匹配:re.match函数

  • 函数语法

    re.match(pattern, string, flags=0)
    参数 描述
    pattern 匹配的正则表达式
    string 要匹配的字符串。
    flags 标志位，用于控制正则表达式的匹配方式
    修饰符 描述
    re.I 使匹配对大小写不敏感
    re.L 做本地化识别（locale-aware）匹配
    re.M 多行匹配，影响 ^ 和 $
    re.S 使 . 匹配包括换行在内的所有字符
    re.U 根据Unicode字符集解析字符。这个标志影响 \w, \W, \b, \B.
    re.X 该标志通过给予你更灵活的格式以便你将正则表达式写得更易于理解。

  • 返回:

    • 匹配成功 re.match 方法返回一个匹配的对象，否则返回 None。
    • 可使用 group(num) 或 groups() 来获取匹配表达式。

  • 示例:

    content = 'Hello 100 World!'
res = re.match(r'^He.*?(\d+)\s.*!$', content)
print(res.group(1))

    输出:

    100
查找

  • 查找第一个:

    res = re.search(r'[\d]{3}', content)

  • 查找所有:

    res = re.findall(r'[\d]{3}', content)
封装匹配方式
  • pattern = re.compile('Xi.*?(\d+)\s.*s',re.S)

安装与导入

  • get_cookies()获取 cookies

数据存储

  • pymysql: mysql操作

    • 操作:

      • 安装: pip install PyMySQL
      • 导入:import pymysql
      • 创建连接:
      conn = pymysql.connect(
    host='localhost',		# 主机名（或IP地址）
    port=3306,				# 端口号，默认为3306
    user='root',			# 用户名
    password='password',	# 密码
    charset='utf8mb4'  		# 设置字符编码
)

      • mysql服务信息: conn.get_server_info()

      • 创建游标对象: cursor = conn.cursor()

      • 选择数据库: conn.select_db("table_name")

      • 执行操作: cursor.execute('SELECT * FROM table_name')

      • 获取查询结果，返回元组: result : tuple = cursor.fetchall()

      • 关闭游标和连接: cursor.close() , conn.close()

      • 数据更改后(insert, update, delete), 需提交更改: conn.commit()

      • 自动提交更改: 在创建连接时, 加上参数autocommit=True

      • 错误处理, 回滚操作: conn.rollback()

        try:
    # 执行一些操作
    ...
    conn.commit()
except Exception as e:
    print(f"发生错误：{e}")
    conn.rollback()
finally:
    cursor.close()
    conn.close()

    • 实例:

      import requests
from bs4 import BeautifulSoup
import pymysql


conn = pymysql.connect(
    host='localhost',
    port=3306,
    user='root',
    password='root',
    charset='utf8mb4'
)

print(conn.get_server_info())
conn.select_db('pymysql')
cur = conn.cursor()

headers = {
    "User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/121.0.0.0 Safari/537.36 Edg/121.0.0.0"
}
for start_num in range(0, 250, 25):
    response = requests.get(f"https://movie.douban.com/top250?start={start_num}", headers=headers)
    content = response.text
    soup = BeautifulSoup(content, "html.parser")
    title = soup.find_all("div", {"class": "info"})
    for info in title:
        for hd in info.find_all("div", {"class": "hd"}):
            name = hd.find("span", {"class": "title"}).string
            cur.execute(f"insert into douban (name) values ('{name}')")
            conn.commit()
